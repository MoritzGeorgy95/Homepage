El Pollo Loco

JavaScript | CSS | HTML

El Pollo loco ist ein Jump & Run Spiel, in dem Pepe Münzen und Tabasco-Salsa sammeln muss, um gegen verrückte Hennen zu kämpfen. Erlebe aufregende Sprünge, Läufe und Würfe, während du Pepe auf seiner Mission begleitest. Die Spiellogik basiert auf Objektorientierung und einem rekursiven Animation Frame.
